# byt
> Firebase-powered link shortener.

---

## i. Installation
1. Clone the repository;
2. Edit and rename `sample.env` to `.env` to match your Firebase's app credentials;
3. Fire the app with `yarn start`.

---

## ii. Features
- URL validator
- No duplicates
